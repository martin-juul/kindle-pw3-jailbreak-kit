# KUAL Extensions

Extensions have been prepared for easing the install. Files can be verified with md5/sha sum. The list below provides the versions for each.

+ BatteryStatus (kual-batterystatus-1.1.N-r16252)
  + Replaces the ellipsis with the actual battery status
+ GNU AWK (kual-gawk-1.5.N-r16737)
  + Speeds up KUAL's text parsing.
+ Helper (kual-helper-0.5.N-r16746)
  + Block OTA and other stuff

## Install

1. Connect Kindle to computer
2. Copy contents of extensions to Kindle -> extensions
3. Eject Kindle
4. Unplug USB
5. On your Kindle, open the KUAL document
6. Press KUAL
7. Press "Install gawk binary"
8. Launch KUAL again
9. Press Helper+
10. Press Prevent OTA Updates
